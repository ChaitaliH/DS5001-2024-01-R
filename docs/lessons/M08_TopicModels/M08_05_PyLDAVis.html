PyLDAVis

Course:   DS 5001
Module:   08 Topic Models
Topic:    PyLDAVis
Author:   R.C. Alvarado
Date:     16 March 2023

Purpose: Demonstrate the use of PyLDAvis, an interactive topic model visualization tool adapted from the R package LDAvis.

Setup

Config

import configparser
config = configparser.ConfigParser()
config.read("../env.ini")
output_dir = config['DEFAULT']['output_dir']
data_prefix = 'austen-melville'

Import

import pandas as pd
import numpy as np
import pyLDAvis
# import pyLDAvis.sklearn
import pyLDAvis.lda_model
import pickle

Import Data

lda_engine = pickle.load(open(f'{output_dir}/{data_prefix}-lda_engine.pickle', 'rb'))
count_engine = pickle.load(open(f'{output_dir}/{data_prefix}-count_engine.pickle', 'rb'))
count_model = pickle.load(open(f'{output_dir}/{data_prefix}-count_model.pickle', 'rb'))

Visualize

pyLDAvis.enable_notebook()
# pyLDAvis.sklearn.prepare(lda_engine, count_model, count_engine)
pyLDAvis.lda_model.prepare(lda_engine, count_model, count_engine)
